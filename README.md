# MiniKartsGeta
This is a variation of Micro-game karts game by Unity, Which includes a Kart customization and also two game modes:<br>
-Against AI<br>
-Count Down<br>
Plus you can find three mod pads, Boost, Slowdown and jump Powerups<br>
